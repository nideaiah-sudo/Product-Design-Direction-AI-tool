# standard-parts-ai 功能介绍

> 离线标准件（螺丝）STEP 生成工具：PDF 规格书 / 照片 → 3D STEP 模型

## 核心功能

### 1. 双路径智能识别

- **PDF 规格书解析**：提取文字、工程图公差、文件名信息；
- **照片 OCR 识别**：通过 Tesseract 识别螺丝照片中的文字；
- **路径完全隔离**：PDF 与照片各自维护独立的解析逻辑，互不干扰，准确率更高。

### 2. 多种螺丝类型支持

| 头型 | 驱动 | 尾部 |
|---|---|---|
| 内六角杯头 | 内六角 | 平尾 / 尖尾 |
| 盘头 | 十字 | 平尾 / 尖尾 |
| 圆头 | 十字 | 平尾 / 尖尾 |
| 沉头 | 十字 | 平尾 / 尖尾 |

### 3. 参数可手动修正

GUI 新增 **“参数修正”** 区域：

- 螺纹大径
- 螺距
- 螺杆长度
- 头部直径
- 头部高度
- 六角对边 / 十字槽号

**使用方式**：

1. 识别结果自动填入参数框；
2. 点击 **“调整”** 解锁参数框；
3. 手动修改错误参数；
4. 点击 **“按标准重置”** 可一键恢复 ISO 标准尺寸；
5. 点击 **“生成 STEP”** 使用修正后的参数生成模型。

### 4. OCR 智能兜底

照片识别具备多层容错机制：

- 多 `M<大径>` 候选择优，自动排除 OCR 噪声；
- 螺距-大径一致性校验，例如识别到 0.5mm 螺距时自动对应 M3；
- 标题误读兜底，例如 `M4*6` 被误读成 `M4*33` 时，用图中真实 `6mm` 修正长度；
- 头径/头高兜底，避免把螺杆长度误当头部直径。

### 5. ISO / DIN / GB 标准尺寸自动补全

- 内置 ISO 4762 / DIN 912 内六角杯头、ISO 7045 盘头、ISO 7380 圆头、ISO 7997 沉头等标准尺寸表；
- 支持 M1.2 ~ M10 规格；
- 自动匹配对应国标号（如 GB/T 70.1-2008、GB/T 818-2016 等）。

### 6. 真实三角螺纹 3D 模型

- 基于 CadQuery 生成带真实三角螺纹的螺丝几何；
- 导出通用 STEP 文件，可直接导入主流 CAD 软件。

### 7. 图形界面 + 命令行双入口

- **GUI**：双击 `standard-parts-ai.exe` 打开；
- **CLI**：`python -m standard_parts_ai "输入.pdf/图片" --out-dir STEP/`，支持批量通配符。

## 离线可用

- 无需联网；
- 打包为独立 exe，拷贝 `dist\standard-parts-ai` 文件夹即可运行；
- 内置 Tesseract OCR 引擎。
